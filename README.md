# Movie Recommendation System

This project implements a content-based recommendation system using TF-IDF and cosine similarity.

Dataset:
TMDB Movie Dataset
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

Features:
• Data preprocessing
• Text vectorization using TF-IDF
• Cosine similarity based recommendations
• Streamlit web application
• Deployment on Render

How it works:
1. Movie descriptions are cleaned
2. TF-IDF converts text to vectors
3. Cosine similarity computes similarity
4. Top similar movies are recommended

Tech Stack:
Python
Pandas
Scikit-learn
Streamlit
